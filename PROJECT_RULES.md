# PROJECT_RULES.md

## Missão
Construir software simples, seguro, sustentável e fácil de manter.

## Regra de Ouro
A melhor solução é a mais simples que resolve o problema corretamente.

## Checklist
- Entenda o problema
- Reutilize antes de criar
- Evite dependências desnecessárias
- Evite overengineering
- Documente decisões importantes
