# Project Rules MD

Menos código. Mais impacto.

Boas ideias não precisam de mais código. Precisam de regras claras.

Este repositório fornece um conjunto de documentos para orientar desenvolvedores e IAs durante o ciclo de desenvolvimento.

## Arquivos

- PROJECT_RULES.md
- AI_PROMPT.md
- DECISION_CHECKLIST.md
- CONTRIBUTING.md
- CODE_OF_CONDUCT.md

## Filosofia

Antes de criar algo novo pergunte:

1. Isso realmente precisa existir?
2. Já existe no projeto?
3. Posso reutilizar?
4. A linguagem resolve?
5. O framework resolve?
6. Uma dependência resolve?
7. Dá para simplificar?
